# Python Values and Data Types

python code :

print(10)

print(10) : called-> function call

print() : called-> function

10 : called-> expression
____________
and every expression has on value ,which has a type:

| Data Type | Example             |
| --------- | ------------------- |
| int       | `10`                |
| float     | `3.14`              |
| complex   | `2+3j`              |
| str       | `"hello"`           |
| bool      | `True`              |
| list      | `[1,2,3]`           |
| tuple     | `(1,2)`             |
| set       | `{1,2,3}`           |
| dict      | `{"name":"Denver"}` |
| NoneType  | `None`              |

---

These data types form the foundation of Python programming and are used to represent different kinds of information in a program.
